# Math Race Junior

เกมคณิตศาสตร์แบบผลัดกันเล่นบน iPad เครื่องเดียว สำหรับเด็ก ป.1–ป.2 รองรับผู้เล่น 1–4 คน ทุกคนได้รับจำนวนตาเท่ากันและไม่มีโบนัสจากความเร็ว

## ฟีเจอร์ใน MVP

- เล่นคนเดียวหรือแข่งขัน 2–4 คนบนเครื่องเดียว
- เลือกการบวก การลบ หรือแบบผสม
- ระดับไม่เกิน 10, ไม่เกิน 20, สองหลักไม่ทด/ไม่ยืม และสองหลักมีทด/มีการยืม
- ตัวเลือกคำตอบ 4 ตัวเลือก พร้อมตัวช่วยภาพ
- ใช้ตัวช่วยแล้วตอบถูกเดิน 1 ช่อง ตอบเองถูกเดิน 2 ช่อง
- สรุปคะแนน ความแม่นยำ และจำนวนครั้งที่ใช้ตัวช่วย
- บันทึกการตั้งค่าล่าสุดในเครื่อง
- PWA และ offline cache สำหรับติดตั้งบนอุปกรณ์ที่รองรับ
- ไม่มีบัญชีผู้ใช้ โฆษณา หรือติดตามข้อมูลเด็ก

## เริ่มใช้งาน

โปรเจกต์ไม่มี dependency ภายนอก ใช้ Node.js 20 ขึ้นไป:

```bash
npm start
```

จากนั้นเปิด `http://127.0.0.1:4173`

ถ้าจะเปิดจาก iPad ที่อยู่ Wi-Fi เดียวกัน บน PowerShell ให้รัน `$env:HOST="0.0.0.0"; npm start` แล้วเปิดผ่าน IP ของคอมพิวเตอร์ พฤติกรรมติดตั้ง PWA/offline บนอุปกรณ์จริงควรตรวจอีกครั้งหลังนำขึ้น HTTPS เช่น GitHub Pages

## ตรวจคุณภาพ

```bash
npm test
npm run check
```

ชุดทดสอบจะสุ่มโจทย์หลายพันข้อเพื่อตรวจว่า:

- ไม่มีคำตอบติดลบ
- มีตัวเลือก 4 ค่าและมีคำตอบถูกเพียงค่าเดียว
- โจทย์ไม่เกินช่วงที่เลือก
- โจทย์สองหลักไม่ทด/ไม่ยืมเป็นไปตามกติกา
- โจทย์มีทด/มีการยืมเกิดการทดหรือยืมจริง

## โครงสร้างโปรเจกต์

```text
├── assets/                  ไอคอนแอป
├── docs/                    คู่มือทดสอบกับเด็กและครู
├── scripts/dev-server.mjs   เซิร์ฟเวอร์สำหรับพัฒนา
├── src/app.js               หน้าจอและ state ของเกม
├── src/question-engine.js   กฎสร้างโจทย์และตัวเลือก
├── tests/                   automated tests
├── index.html
├── styles.css
└── service-worker.js
```

## หลักการของเกม

- การแข่งขันหนึ่งรอบจบตาของผู้เล่นทุกคนก่อนประกาศผู้ชนะ
- ผู้เล่นทุกคนได้จำนวนโจทย์เท่ากัน
- ความเร็วไม่เพิ่มคะแนน
- คำใบ้เป็นเครื่องมือเรียนรู้ ไม่ตัดสิทธิ์ในการตอบ
- ชื่อและผลการแข่งขันอยู่ในอุปกรณ์เท่านั้น

ก่อนขยายฟีเจอร์ ให้ทดสอบ vertical slice นี้ตาม [คู่มือ playtest](./docs/playtest-guide.md) และตัดสินใจจากพฤติกรรมเด็กจริง
